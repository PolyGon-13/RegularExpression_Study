## 정규 표현식
도서 : 손에 잡히는 10분 정규 표현식
</br></br>

### 기본 메타 문자
|표현식|기능|
|:---:|:---:|
|.|모든 문자와 일치|
|\||왼쪽이나 오른쪽과 일치|
|[]|문자 집합 구성원 중 하나와 일치|
|[^]|문자 집합 구성원을 제외하고 일치|
|-|범위 정의|
|\|다음에 오는 문자를 이스케이프|

### 수량자
|표현식|기능|
|:---:|:---:|
|*|문자가 없거나 하나 이상 연속하는 문자 찾기|
|+|문자 하나 이상 찾기|
|?|문자 없거나 하나인 문자 찾기|
|{n}|요소와 n번 일치|
|{m,n}|요소와 m에서 n번 일치|
|{n,}|요소와 n번 이상 일치|
